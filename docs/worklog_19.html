Week 19 (September 23rd - September 27th)

October 3rd, 2024 by María Busto Ramos

Here are the tasks I have been working on during this 19th week of internship at JdeRobot:

  • I have finished implementing the password reset fix. I have tested it with university mail and Gmail mail on D1 and it already works correctly. I have added a note about the solution to Unibotics' D1 documentation and the issue. I have opened Pull Request.

  • I have continued to check the new user registration error with university emails. It seems as if two clicks on the activation link are being detected in the case of university emails, because the account is being activated, but you see the registration page next. To fix this, we have added a new view so that if the user has already activated the account but uses the activation link, it will warn you that the account is already activated. We tested it on D1 and it works with four types of mail, including university. Therefore, we have opened a Pull Request.

  • I have started moving the efficacy evaluator from the old version of Obstacle Avoidance to the new newmanager version. Errors have appeared during the evaluation process that need to be fixed and HTML elements in the viewer have been misplaced.
